# AuresPédia — Image Generation Prompts
> One prompt per image slot in the website. Use ChatGPT (DALL-E 3) or Midjourney.
> Recommended: paste prompt exactly, add **"photorealistic, no watermark, no text overlay"** at the end of every prompt.
> Recommended sizes: **1280×720** for banners/cards — **400×400** for team headshots.

---

## GLOBAL STYLE RULES (apply to ALL images)
- Setting: Algeria — CHU Batna, Université de Batna 2, or Algerian urban environments
- People: realistic Algerian skin tones, natural facial features
- Women: natural mix — some with hijab (various colors), some without. Never 100% hijab, never 0%
- Men: white lab coat or smart casual. No stereotypes
- Lighting: warm natural or soft clinical light. No harsh flash
- Mood: professional, calm, reassuring. No fear or pain visible in patients

## TIPS FOR DALL-E 3 (ChatGPT)

- Use **DALL-E 3** not DALL-E 2 — much more realistic
- Always end with: `photorealistic, no watermark, no text overlay`
- If face looks odd: add `natural facial features, realistic Algerian skin tones`
- If hijab looks wrong: add `properly worn Islamic headscarf, fabric naturally draped`
- For consistent style: start a single ChatGPT conversation and generate all images in sequence
- Regenerate any image that looks AI-obvious — usually 1–2 tries gives a good result
- **Landscape images** (banners/cards): set image size to **1792×1024** in DALL-E settings
- **Square images** (team headshots): set size to **1024×1024**
